Generador y Validador de CAPTCHA (PHP + JS + Python)
Descripción del Proyecto

Sistema de generación y validación de CAPTCHA desarrollado principalmente con PHP y JavaScript, enfocado en reforzar la seguridad de formularios web.
El proyecto genera una imagen CAPTCHA dinámica desde el servidor y valida la entrada del usuario antes de permitir el envío del formulario.

Adicionalmente, se incorpora Python como apoyo para pruebas, automatización y generación de datos simulados, reforzando la lógica de validación del sistema.

Características del Proyecto

Generación dinámica de CAPTCHA desde el servidor.

Validación del código CAPTCHA en backend.

Refresco automático del CAPTCHA sin recargar la página.

Manipulación del DOM con JavaScript.

Separación clara entre lógica frontend y backend.

Uso de Python para pruebas de validación y apoyo en lógica de seguridad.

¿Qué se aprende en este proyecto?

Comunicación entre frontend y backend.

Manejo de sesiones para validación de datos.

Generación de imágenes dinámicas en servidor.

Validación de formularios segura.

Uso combinado de PHP, JavaScript y Python.

Automatización básica de pruebas con Python.

Tecnologías Utilizadas

Cómo ejecutar el proyecto

Clona el repositorio.

Coloca el proyecto en tu servidor local (XAMPP, Laragon o similar).

Accede desde el navegador a index.php.

Ingresa el CAPTCHA generado para validar el formulario.

(Opcional) Ejecuta los scripts de Python para pruebas de validación.
